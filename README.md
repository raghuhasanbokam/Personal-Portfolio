# Personal-Portfolio

Live: `https://raghuhasanbokam.vercel.app/`

Source: `https://github.com/raghuhasanbokam/Personal-Portfolio`

## Tech stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- shadcn/ui (Radix UI)
- Vercel (deploy) + Vercel Analytics

## Run locally

```bash
npm install
npm run dev
```
